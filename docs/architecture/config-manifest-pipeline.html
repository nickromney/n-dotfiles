Catalog module

Configuration bundles to install manifests

Configuration Bundles are YAML source. Install Manifests are the generated interface consumed by install and update modules.

Pipeline

flowchart LR
  Configs[_configs/*.yaml] --> Make[Makefile profile selection]
  Make --> Generator[scripts/generate-install-manifests.sh]
  Generator --> Brewfile[Brewfile]
  Generator --> Arkade[arkade.tsv]
  Generator --> Metadata[metadata.json]
  Metadata --> InstallLib[scripts/install-lib.sh]
  Brewfile --> BrewBundle[brew bundle]
  Arkade --> ArkadeBatch[arkade get batch]

Configuration Bundle

A bundle under _configs/ names tools, managers, types, checks, dependencies, update behavior, descriptions, and documentation URLs.

Install Manifest

The manifest interface strips YAML shape away from execution. The engine reads rows, not config files.

Leverage

New managers need one manifest adapter. New bundles need no execution changes when they fit existing manager/type rows.
